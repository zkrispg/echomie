# EchoMie 前期基础设施与合规准备说明

| 文档用途 | 域名、备案、服务器、HTTPS 等上线前准备工作的说明与核对表 |
|----------|----------------------------------------------------------|
| 适用对象 | 项目负责人、运维、备案联系人 |
| 关联系统 | EchoMie Web / 微信小程序（共用后端与域名策略） |

---

## 1. 总体顺序建议

建议按以下顺序推进，可减少返工：

1. 确定 **云账号与主体**（个人或企业），后续备案主体需与小程序主体策略一致。  
2. 购买 **域名** 并完成 **实名认证**。  
3. 购买 **服务器**（建议包年包月 ≥ 3 个月，满足多数地区备案要求）。  
4. 完成 **ICP 备案**（备案期间网站勿开放违规内容，按接入商要求关站或展示备案中页面）。  
5. 备案通过后配置 **域名解析** 指向服务器公网 IP。  
6. 在服务器上部署应用并申请 **HTTPS 证书**（Let's Encrypt 或云证书）。  
7. 微信小程序后台配置 **request 合法域名**（仅 HTTPS + 已备案域名）。

## 2. 域名相关

| 工作项 | 说明 | 核对 |
|--------|------|------|
| 注册域名 | 在阿里云等注册商购买，`.com` / `.cn` / `.top` 等均可；**小程序 request 域名需合规且可在微信后台添加** | □ |
| 域名实名 | 域名持有者需完成实名，未完成实名可能影响解析与备案 | □ |
| DNS 解析 | 备案通过且服务器就绪后，添加 **A 记录** 指向 ECS 公网 IP（可含 `www`） | □ |
| 解析生效 | 各地 DNS 缓存不同，一般数分钟至 48 小时 | □ |

---

## 3. ICP 备案相关

| 工作项 | 说明 | 核对 |
|--------|------|------|
| 备案主体 | 个人或企业证件信息与云账号、备案系统填报一致 | □ |
| 服务器要求 | 常见要求：**包月剩余时长 ≥ 3 个月**（以接入商/管局规则为准）；按量实例可能不符合 | □ |
| 接入信息 | 填写 ECS 实例 ID、网站名称、网站内容说明等 | □ |
| 管局审核 | 约 7～20 个工作日不等，期间保持电话畅通 | □ |
| 备案号展示 | 网站首页或底部展示 **备案号**，并链接至工信部备案查询（按当地规范） | □ |

## 4. 服务器（ECS）相关

| 工作项 | 说明 | 核对 |
|--------|------|------|
| 规格选型 | EchoMie 当前栈：Docker + PostgreSQL + Redis + 后端 + Nginx；**2 核 4G** 可起步，用户增多再升配 | □ |
| 操作系统 | 常见：**Ubuntu 22.04** 或 Alibaba Cloud Linux | □ |
| 公网 IP | 分配弹性公网 IP；记录 IP 供解析与 SSH | □ |
| 安全组 | 放行 **22**（SSH，建议仅办公网）、**80/443**（HTTP/HTTPS）；数据库、Redis **勿对公网开放** | □ |
| 磁盘与备份 | 系统盘 + 数据盘规划；重要数据与数据库制定 **备份策略** | □ |
| 计费方式 | 长期运行建议 **包年包月**；短期测试可按量（注意备案时长要求） | □ |

## 5. 部署与 HTTPS

| 工作项 | 说明 | 核对 |
|--------|------|------|
| 环境变量 | 生产环境配置 `DATABASE_URL`、`REDIS_URL`、`JWT_SECRET_KEY`、`DASHSCOPE_API_KEY`、`STORAGE_BASE_PATH` 等 | □ |
| 反向代理 | Nginx 将 443 转发至前端容器，将 `/api`、`/static` 等转发至后端 | □ |
| SSL 证书 | 使用 Certbot 或云厂商证书；配置 **自动续期** | □ |
| 仅 HTTPS | 小程序要求 API 为 **HTTPS**；HTTP 可做 301 跳转至 HTTPS | □ |

---

## 6. 与微信小程序的衔接

| 工作项 | 说明 | 核对 |
|--------|------|------|
| 合法域名 | 微信公众平台 → 开发 → 服务器域名，添加 **https://你的域名**（不含路径与端口） | □ |
| 备案与 HTTPS | 未备案域名通常无法作为小程序合法域名；**不支持裸 IP** | □ |
| AppSecret | 仅保存在 **服务端** 环境变量，勿写入前端仓库 | □ |

---

## 7. 风险与常见问题

| 问题 | 处理建议 |
|------|----------|
| 备案提示服务器时长不足 | 续费至 ≥ 3 个月或更换符合要求的包月实例 |
| 域名显示未实名 | 在域名控制台完成模板实名并关联域名 |
| 解析后仍无法访问 | 检查安全组、本机防火墙、Nginx 是否监听 80/443 |
| Docker 构建慢 | 使用国内镜像源；大文件构建可用 `nohup` 后台执行避免 SSH 断开 |
